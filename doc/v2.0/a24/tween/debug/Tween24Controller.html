Tween24Controller(x:Number = 0, y:Number = 0, preset:Array = null)
Tween24 のグローバルタイムスケールを制御するコントローラーインスタンスを生成します。 プリセットに設定されているものは自動的にキーボートの数値キーにも割り当てられるので、キーボードでの操作も可能になります。 キーボードでの操作は、キーを押している間のみタイムスケールが変更され、離すと元のスケールに戻ります。 また、addKey()を使用することで任意のキーに割り当てることも可能です。

public function Tween24Controller(x:Number = 0, y:Number = 0, preset:Array = null)

Tween24 のグローバルタイムスケールを制御するコントローラーインスタンスを生成します。 プリセットに設定されているものは自動的にキーボートの数値キーにも割り当てられるので、キーボードでの操作も可能になります。 キーボードでの操作は、キーを押している間のみタイムスケールが変更され、離すと元のスケールに戻ります。 また、addKey()を使用することで任意のキーに割り当てることも可能です。

Parameters
x:Number (default = 0) — 配置するX座標
 
y:Number (default = 0) — 配置するY座標
 
preset:Array (default = null) — コントローラーに配置する倍速値（デフォルト-[0.5, 1, 1.5, 2, 5, 10, 24] ）
